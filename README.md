# praktikum-rpl-a-02

Halo MBG

Aplikasi Monitoring MBG (Makanan Bergizi Gratis)

Tujuan
Tujuan dari aplikasi ini adalah untuk memudahkan masyarakat dalam mengetahui lokasi dan jadwal pembagian makanan bergizi gratis yang disediakan oleh pemerintah atau organisasi sosial. Dengan aplikasi ini, diharapkan dapat meningkatkan kesadaran masyarakat tentang pentingnya makanan bergizi dan membantu mereka mendapatkan akses ke makanan sehat secara gratis.

Fitur Utama

1. Peta Lokasi: Menampilkan peta interaktif yang menunjukkan lokasi-lokasi pembagian makanan bergizi gratis di sekitar pengguna.
2. Jadwal Pembagian: Menyediakan informasi jadwal pembagian makanan bergizi gratis, termasuk tanggal, waktu, dan lokasi pembagian.
3. Notifikasi: Mengirimkan notifikasi kepada pengguna tentang pembagian makanan bergizi gratis yang akan datang di area mereka.
4. Informasi Nutrisi: Menyediakan informasi tentang nilai gizi dari makanan yang dibagikan, sehingga pengguna dapat membuat pilihan yang lebih sehat.
5. Feedback Pengguna: Memungkinkan pengguna untuk memberikan feedback tentang pengalaman mereka dengan pembagian makanan bergizi gratis, sehingga dapat membantu penyelenggara meningkatkan layanan mereka.

Teknologi yang Digunakan

1. Frontend: React Native untuk pengembangan aplikasi mobile yang dapat berjalan di platform iOS dan Android.
2. Backend: Node.js dengan Express untuk mengelola API dan database.
3. Database: MongoDB untuk menyimpan data lokasi, jadwal, dan informasi nutrisi.
4. Peta: Google Maps API untuk menampilkan peta interaktif.
5. Notifikasi: Firebase Cloud Messaging untuk mengirimkan notifikasi kepada pengguna.

Cara Menggunakan Aplikasi

1. Unduh dan instal aplikasi dari App Store atau Google Play Store.
2. Buka aplikasi dan buat akun pengguna dengan memasukkan informasi yang diperlukan.
3. Setelah masuk, pengguna dapat melihat peta lokasi pembagian makanan bergizi gratis di sekitar mereka.
4. Klik pada lokasi untuk melihat jadwal pembagian dan informasi nutrisi dari makanan yang dibagikan.
5. Aktifkan notifikasi untuk menerima pemberitahuan tentang pembagian makanan bergizi gratis yang akan datang.
6. Berikan feedback setelah menggunakan layanan untuk membantu meningkatkan kualitas pembagian makanan bergizi gratis.

Kesimpulan

Aplikasi Monitoring MBG (Makanan Bergizi Gratis) merupakan solusi yang efektif untuk membantu masyarakat mendapatkan akses ke makanan sehat secara gratis. Dengan fitur-fitur yang lengkap dan teknologi yang digunakan, aplikasi ini dapat meningkatkan kesadaran masyarakat tentang pentingnya makanan bergizi dan memudahkan mereka dalam menemukan lokasi pembagian makanan bergizi gratis di sekitar mereka.

Terima kasih telah menggunakan aplikasi Monitoring MBG! Kami berharap aplikasi ini dapat memberikan manfaat bagi masyarakat dalam mendapatkan akses ke makanan bergizi gratis. Jangan ragu untuk memberikan feedback agar kami dapat terus meningkatkan layanan kami.
